# 12 Step Design Process Primer

An HTML slideshow primer for the **Purdue University Fort Wayne Center of Excellence in Systems Engineering**, covering a 12-step systems engineering design process used when a customer provides "requirements."

**Live site:** https://flylow3d.github.io/12-step-design-primer/

## Authors
- Prof. David S. Cochran, Ph.D.
- Joseph J. Smith

## Running locally
Open `index.html` in any modern browser — no build step required. PDF export uses `html2canvas` and `jsPDF` from a CDN, so it needs an internet connection.

## Controls
- **Prev / Next** buttons or arrow keys to navigate
- **Present ▾** dropdown for fullscreen mode (Standard 4:3 or Widescreen 16:9)
- **Download PDF** to export all slides
- Press **F** to toggle present mode, **Esc** to exit
